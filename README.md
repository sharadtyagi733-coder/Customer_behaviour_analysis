# Data Analytics End-to-End Project

## Overview

This project demonstrates a complete Data Analytics workflow, covering data extraction, cleaning, analysis, visualization, and reporting. The objective is to transform raw data into actionable business insights using Python, SQL, and Power BI.

The project includes:

* Data loading and preprocessing using Python
* Exploratory Data Analysis (EDA)
* Data cleaning and transformation
* SQL-based data analysis
* Interactive Power BI dashboard creation
* Business insights report generation
* Presentation creation using Gamma

---

## Dataset

The dataset contains business-related records used for analytical purposes. It includes multiple attributes such as customer information, transactions, sales performance, product details, and other relevant metrics.

### Dataset Features

* Structured tabular data
* Multiple dimensions and measures
* Suitable for trend, performance, and KPI analysis

### 3. Data Cleaning

## How to Run

### Python Analysis

1. Clone the repository.
2. Install required packages:

```bash
pip install pandas numpy matplotlib seaborn
```

3. Open Jupyter Notebook or VS Code.
4. Run the notebooks in the Python folder.

### SQL Analysis

1. Import the cleaned dataset into:

   * PostgreSQL
   * MySQL
   * SQL Server

2. Execute the SQL scripts available in the SQL folder.

### Power BI Dashboard

1. Open the `.pbix` file in Power BI Desktop.
2. Refresh the data connection.
3. Explore the dashboard and visualizations.
